Question-1: What are the different ways to select an element in the DOM?

answer:

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

    • Question-2: What is the difference between innerHTML, innerText, and textContent ?

    Answer:

  • innerHTML: innerHTML return Html tags
  • innerText: innerText return inside tags content
  • innerContent: innerContent return inside tags content and hidden text contet from child element

    • Question-3: What is event delegation in the DOM?

    Answer: প্যারেন্ট উপাদানে ইভেন্ট লাগানো এবং সেই প্যারেন্টের ভিতরের সমস্ত চাইল্ড উপাদানের ইভেন্ট হ্যান্ডলিং করা।

    Question-4: What is event bubbling in the DOM?

    Answer: যখন কোনো ইভেন্ট (যেমন, ক্লিক) একটি উপাদানে ঘটে, তখন সেটি প্যারেন্ট উপাদানগুলোতে পৌঁছাতে থাকে, একে একে উপরের দিকে।

    Question-5: How do you create, add, and remove elements using JavaScript?

    Answer:

  • Create an element: Use document.createElement()
  • Add an element: Use appendChild()
  • Remove an element: Use removeChild()
